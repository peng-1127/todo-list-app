## 📸 使用方法

### 本地运行
```bash
# 方式 1: 直接打开
start index.html

# 方式 2: 本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 部署到 GitHub
```bash
# 1. 在 GitHub 创建新仓库
# 2. 执行以下命令
git remote add origin https://github.com/YOUR_USERNAME/todo-list-app.git
git branch -M main
git push -u origin main
```

### 部署到 GitHub Pages
1. 推送到 GitHub
2. Settings → Pages
3. 选择 main 分支
4. 访问 https://YOUR_USERNAME.github.io/todo-list-app
